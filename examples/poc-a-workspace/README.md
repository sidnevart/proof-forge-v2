# Example Developer Workspace

This folder is a sample of what a developer would hand to the Proof-Forge agent when starting a learning session. It contains:

- `notes/` — markdown notes from the study session
- `code/` — code artifacts written or explored during learning

## How to use

Start a Claude session with the MCP plugin connected and say:

> "Start topic: JavaScript Closures. user_id: <your-user-id>"

Then:

> "Analyze my code: [paste contents of counter.js]"

> "I have these notes: [paste contents of closures.md]"

> "Build capsule"

> "Run review"
