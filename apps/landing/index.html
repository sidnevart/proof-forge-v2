Полный цикл обучения

Четыре шага, которые ведут от «не знаю» к «объясню другому».

01 · learn

Изучай

Ментор объясняет тему простым языком со схемами и даёт по-настоящему сложную практику — калибрует под тебя.

02 · capsule

Фиксируй

Капсула знаний: Replay, концепт-карта, слабые места, вопросы для ревью — возвращайся к ней в любой момент.

03 · cards

Повторяй

Карточки по алгоритму SM-2: интервал растёт по мере того как ты вспоминаешь увереннее.

04 · review

Проверяй

Интерактивный quiz: hint-лесенка вместо готового ответа, struggle-check на edge-кейсы.

Не лозунг, а измеримый результат

🟥🟨🟩🟦

Прогресс к эксперту

Каждый концепт проходит уровни: не начат → узнаю → применяю → объясняю. «Эксперт по теме» = все концепты на 🟦. Видно что осталось доработать.

⚖️

Честная сложность

Задание сложное не по объёму, а по нагрузке: edge-кейсы, конфликтующие требования, обоснование trade-off. Легко далось — планка поднимается.

🔁

Интервальное повторение

SM-2 разводит карточки во времени так, чтобы ты повторял ровно перед тем как забыть. Минимум усилий — максимум удержания.

🌐

С любого устройства

Профиль привязан к email — учись с ноутбука, продолжай с другого. Весь прогресс в облаке.

🧠

Агент — это мозг

Всё мышление в сессии Claude. Бэкенд — чистое хранилище. Никаких чужих LLM-вызовов с твоими данными.

Там где ты работаешь

Claude Desktop и Claude Code. Никаких новых приложений — учишься прямо в чате, рядом с кодом.

Установка за минуту

  1. шаг 1 · установи плагин
    uv tool install "git+https://github.com/sidnevart/proof-forge-v2.git#subdirectory=apps/mcp-server"
  2. шаг 2 · добавь в конфиг Claude Desktop
    {
  "mcpServers": {
    "proof-forge": {
      "command": "proof-forge-mcp",
      "env": { "BACKEND_URL": "https://api.proof-forge.ru" }
    }
  }
}
  3. шаг 3 · перезапусти Claude и пиши
    помоги изучить JavaScript Closures

    При первом запуске Claude спросит email и создаст профиль. Всё.
